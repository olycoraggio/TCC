1. LOADING
	- LOGO
	- CORES
	- FONTE
	
2. LOCALIZAÇÃO
	- BARRA FIXA COM LOGO E NOME DO ESTABELECIMENTO
	- CAMPO COM A NUMERAÇÃO DA MESA
	- CAMPO COM A QUANTIDADE DE PESSOAS
	- CAMPO COM FORM PRA COLOCAR NOME DAS PESSOAS NA MESA
	- BOTÃO PARA ENTRAR
	- BOTÕES DAS REDES SOCIAIS
	
3. CARDÁPIO
	- BARRA FIXA COM OPÇÃO DE VOLTAR A PÁGINA ANTERIOR
	- TITULO DA PÁGINA NO INICIO EM DESTAQUE
	- TABS DE 'PRATOS', 'BEBIDAS' E 'SOBREMESAS'
	- CADA TAB COM A OPÇÃO DE PELO MENOS UM ITEM
	- CADA ITEM COM A VARIAÇÃO DE PREÇO
	- CADA ITEM COM SUA DESCRIÇÃO
	- CADA ITEM COM UM SELETOR DE QUANTIDADE
	- RODAPÉ COM VALOR FINAL DO ESCOLHIDO
	- BOTÃO PARA DIRECIONAR A LISTA DE CARRINHO
	- BOTÃO PARA PAGAMENTO DIRETO
	
4. CARRINHO
	- BARRA FIXA COM TITULO DESTACADO
	- LISTA DE ITENS AGRUPADOS POR CATEGORIA
	- VALOR UNITÁRIO DO ITEM
	- VALOR TOTAL DOS ITENS COM MAIS DE UMA SELEÇÃO
	- BOTÃO DE ALTERAÇÃO DE ITENS GERAL
	- BOTÃO DE FECHAMENTO DE PEDIDO E ENCAMINHAMENTO PARA PÁGINA DE PAGAMENTO
	
5. PAGAMENTO
	- BARRA FIXA DESTACANDO O TITULO DA PAGINA
	- TABS COM OPÇÕES DE PAGAMENTOS
	- DINHEIRO:
		CAMPO PARA PREENCHIMENTO COM O VALOR A SER PAGO E SE NECESSÁRIO TROCO
		BOTÃO DE CONFIRMAÇÃO DO PAGAMENTO
	- PIX:
		ZONA DE EXIBIÇÃO DA CHAVE PIX DO ESTABELECIMENTO
	- CARTÃO:
		ZONA DE PAGAMENTO CONFORME API REQUERIDA DO ESTABELECIMENTO
